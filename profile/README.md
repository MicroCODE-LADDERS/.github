## Product Background

LADDERS¬Æ has a 40+ year history in PLC programming support, evolving from:
- **16-Bit** (1982): DEC PDP-11 Mini-Computers running RT-11 OS
- **32-Bit** (1989): VAX/VMS multi-user datacenter servers
- **64-Bit** (1997): DEC Alpha-AXP hardware with OpenVMS
- **Cloud Service** (Current): Modern cloud-based SaaS platform

## Core Capabilities

- **PLC Compare Reporting**: Generate detailed comparison reports between PLC program versions
- **ControlLogix 5000 Support**: Primary support for Allen-Bradley ControlLogix 5000 PLCs
- **Cloud Accessibility**: Access reports from anywhere
- **Difference Highlighting**: Clear visualization of changes between program versions

## Repository Structure

This organization contains private repositories for:
- **Backend Services**: Cloud infrastructure and PLC data processing
- **Frontend Application**: Web-based user interface
- **PLC Communication**: Protocol handlers and device interfaces
- **Report Generation**: Comparison engine and report formatting
- **Documentation**: Technical specs and API documentation
